## ZOOV

ZOOV = Zustand + Module

用模块化的语法来使用 zustand

### Features

- 😌 舒适的类型推断
- 🪝 拒绝繁琐的 Selector，自动生成状态 hooks
- 🍳 简单的封装，无魔法
- 🧮 在 Zustand 基础上扩展 computed
